# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a 3-way dark mode toggle (Light / Dark / System) that applies across every screen and component in the mobile app.

**Architecture:** NativeWind's class-based dark mode (`darkMode: "class"`) is controlled via `useColorScheme()` from `nativewind`. A `colorScheme` field in `StoredSettings` persists the user's choice. On startup, the root layout reads the stored preference and calls `setColorScheme()`. All 29 screen/component files get `dark:` Tailwind variants. A small `useThemeColors` hook provides hex values for the ~15 inline `style=` props that can't use Tailwind classes.

**Tech Stack:** React Native, Expo, NativeWind v4, Tailwind CSS, AsyncStorage

| `active:bg-zinc-50`     | `dark:active:bg-zinc-800` |

| `bg-green-50`           | `dark:bg-green-950`       |
| `bg-blue-50`            | `dark:bg-blue-950`        |
| `bg-amber-100`          | `dark:bg-amber-950`       |
| `text-base` (unlabeled) | `dark:text-zinc-100`      |

**Rule for `text-base` with no explicit color:** Any `<Text className="text-base">` or `<Text className="text-base font-medium">` etc. that doesn't already have a `text-*` color class needs `dark:text-zinc-100` added — otherwise it renders as black text on dark backgrounds.

**Rule for headings:** `text-3xl font-bold`, `text-xl font-bold`, `text-lg font-semibold`, `text-2xl font-bold`, `text-base font-semibold`, `text-base font-medium` — any Text with font styling but no explicit color class needs `dark:text-zinc-100`.

**Accent colors unchanged:** `bg-blue-500`, `text-blue-500`, `bg-red-500`, `text-red-500`, `bg-green-500`, `text-green-500`, `bg-amber-600`, `text-amber-600` — all keep working on dark backgrounds without modification.

import { useColorScheme } from "nativewind";

export function useThemeColors() {
  const { colorScheme } = useColorScheme();
  const dark = colorScheme === "dark";

  return {
    // Backgrounds
    cardBg: dark ? "#18181b" : "#ffffff",
    subtleBg: dark ? "#27272a" : "#f4f4f5",
    pageBg: dark ? "#09090b" : "#fafafa",

    // Text
    primaryText: dark ? "#fafafa" : "#18181b",
    secondaryText: dark ? "#a1a1aa" : "#71717a",
    tertiaryText: dark ? "#71717a" : "#a1a1aa",

    // Icons
    accentIcon: "#3b82f6",
    mutedIcon: dark ? "#71717a" : "#a1a1aa",
    chevron: dark ? "#71717a" : "#a1a1aa",

- [ ] **Step 1: Update root layout to set color scheme on startup**

In `apps/mobile/app/_layout.tsx`, add the color scheme wiring. The full file should become:

```tsx
import "../global.css";
import { Slot, useRouter, useSegments } from "expo-router";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { AuthProvider, useAuth } from "../hooks/useAuth";
import { WorkoutProvider } from "../hooks/useWorkout";
import { useBootstrap } from "../hooks/useBootstrap";
import { ActivityTracker } from "../components/ActivityTracker";
import { getSettings, hydrateStorage } from "../services/storage";
import { useEffect, useState } from "react";
import { View, ActivityIndicator, Text, Appearance } from "react-native";
import { useColorScheme } from "nativewind";

const queryClient = new QueryClient();

function ColorSchemeProvider({ children }: { children: React.ReactNode }) {
  const { setColorScheme } = useColorScheme();

  useEffect(() => {
    const stored = getSettings().colorScheme;
    if (stored === "system") {
      setColorScheme("system");
    } else {
      setColorScheme(stored);
    }
  }, []);

  return <>{children}</>;
}

function AuthGate() {
  const { isAuthenticated, isLoading } = useAuth();
  const { isBootstrapped, isLoading: isBootstrapping } = useBootstrap();
  const segments = useSegments();
  const router = useRouter();

  useEffect(() => {
    if (isLoading) return;
    const inAuth = segments[0] === "login";
    if (!isAuthenticated && !inAuth) router.replace("/login");
    else if (isAuthenticated && inAuth) router.replace("/(tabs)/me");
  }, [isAuthenticated, isLoading, segments]);

  if (isLoading) {
    return (
      <View className="flex-1 items-center justify-center bg-white dark:bg-zinc-900">
        <ActivityIndicator size="large" />
      </View>
    );
  }

  if (isAuthenticated && isBootstrapping && !isBootstrapped) {
    return (
      <View className="flex-1 items-center justify-center bg-white dark:bg-zinc-900">
        <ActivityIndicator size="large" />
        <Text className="mt-4 text-gray-500 dark:text-zinc-400">Syncing data...</Text>
      </View>
    );
  }

  return (
    <>
      <ActivityTracker />
      <Slot />
    </>
  );
}

export default function RootLayout() {
  const [storageReady, setStorageReady] = useState(false);

  useEffect(() => {
    hydrateStorage().then(() => setStorageReady(true));
  }, []);

  if (!storageReady) {
    return (
      <View className="flex-1 items-center justify-center bg-white dark:bg-zinc-900">
        <ActivityIndicator size="large" />
      </View>
    );
  }

  return (
    <QueryClientProvider client={queryClient}>
      <ColorSchemeProvider>
        <AuthProvider>
          <WorkoutProvider>
            <AuthGate />
          </WorkoutProvider>
        </AuthProvider>
      </ColorSchemeProvider>
    </QueryClientProvider>
  );
}
```
